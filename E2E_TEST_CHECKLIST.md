## 📱 啟動應用

### 後端（已運行）
```bash
cd backend
npm run start:dev
```
- ✅ 運行中：http://localhost:5000/api

### 前端
```bash
cd frontend
npx expo start
```
- 選項：按 `w` (Web), `a` (Android), `i` (iOS)
